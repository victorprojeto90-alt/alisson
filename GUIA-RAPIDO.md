# 🚀 Guia Rápido - AMBISAFE Geotecnologias

## Para Começar

### 1. Criar Conta
- Acesse a landing page
- Clique em **"Teste Grátis 14 Dias"**
- Preencha: Nome, Empresa, E-mail e Senha
- Sem necessidade de cartão de crédito!

### 2. Primeiro Login
- Use o e-mail e senha cadastrados
- Você será direcionado ao Dashboard

### 3. Criar Primeiro Projeto
1. Clique em **"Projetos"** no menu lateral
2. Clique em **"Novo Projeto"**
3. Preencha:
   - Nome do Projeto (ex: "Fazenda Santa Helena")
   - Área em hectares (ex: "1250")
   - Tipo de Amostragem (escolha na lista)
4. Faça upload da planilha (CSV ou XLSX)
5. Clique em **"Criar Projeto"**

### 4. Visualizar Resultados
- Clique no projeto criado
- Veja:
  - **Score AMBISAFE** (0-100)
  - **Indicadores** (Volume, Densidade, Shannon)
  - **Gráficos** interativos
  - **Estatísticas** avançadas

### 5. Gerar Relatórios com IA
- Na tela do projeto, vá em "Relatórios com IA"
- Escolha o tipo:
  - Técnico Completo
  - Executivo
  - Plano de Manejo
- Clique para baixar o PDF

## 📋 Formato da Planilha

Sua planilha deve conter estas colunas:

| Coluna | Descrição | Obrigatória |
|--------|-----------|-------------|
| Parcela | ID da parcela | ✅ Sim |
| Espécie | Nome da espécie | ✅ Sim |
| DAP ou CAP | Diâmetro/Circunferência | ✅ Sim |
| HT | Altura Total (m) | ✅ Sim |
| Outras | Dados adicionais | ❌ Não |

### Exemplo:
```csv
Parcela,Espécie,DAP,HT
1,Ipê,45.2,18.5
1,Jatobá,38.7,16.2
2,Cedro,52.3,21.0
```

## 🎯 Dicas

### Para Melhores Resultados
- ✅ Use dados de campo validados
- ✅ Mantenha consistência nas unidades
- ✅ Verifique valores extremos
- ✅ Documente metodologia

### Score AMBISAFE
- 🟢 **80-100**: Excelente qualidade
- 🟡 **50-79**: Bom, com oportunidades
- 🔴 **0-49**: Necessita revisão

### Interpretação Rápida
- **Erro Amostral < 10%**: Boa representatividade
- **Shannon > 3.0**: Alta diversidade
- **IVI balanceado**: Boa distribuição
